# Simple Meditation🌿

**Simple Meditation** — это минималистичное Android-приложение для осознанности, созданное в мастерской «Дом Бубнов». Оно родилось как протест против перегруженных медитативных «комбайнов» с подписками, рекламой и бесконечным шумом.

> "Стор отклонил это приложение за 'недостаточный функционал'. Мы считаем это лучшим комплиментом: здесь нет ничего лишнего, только вы и ваша тишина."

## ✨ Особенности

* **Чистый звук:** 9 атмосферных ландшафтов (Костер, Лягушки, Волки, Снег и др.) в формате WAv.
* **Умный аудио-движок:** Плавное нарастание звука при старте (2 сек), бесшовные переходы между циклами и глубокое затухание при выходе (3 сек).
* **Библиотека Смыслов:** Короткие карточки с научными фактами и техниками вместо навязчивых аудио-гидов.
* **Полная приватность:** 0 разрешений, 0 трекеров, 0 сбора данных. Работает полностью офлайн.
* **Эстетика:** 4 визуальные темы (Светлая, Темная, Океан, Закат).

## 🛠 Технологии

* **Язык:** Kotlin
* **UI:** Jetpack Compose
* **Media:** Android MediaPlayer с кастомной логикой управления громкостью (Fade In/Out).
* **Архитектура:** Чистый и лаконичный код без тяжелых зависимостей.

## 🚀 Как запустить

1. Клонируйте репозиторий: 
   `git clone https://github.com/KayaYaM/simple-meditation.git`
2. Откройте проект в **Android Studio**.
3. Дождитесь синхронизации Gradle.
4. Соберите APK через `Build > Build Bundle(s) / APK(s) > Build APK(s)`.

## 📦 Установка (для пользователей)

Если вы не разработчик, вы можете скачать готовый `.apk` файл из раздела [Releases](https://github.com/KayaYam/simple-meditation/releases/). 

*Примечание: При установке Android может попросить "Разрешить установку из неизвестных источников". Это стандартная процедура для приложений, распространяемых вне Google Play / RuStore.*
